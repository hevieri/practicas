# 💻 Prácticas de Desarrollo Web

Bienvenidos a mi espacio de exploración web. Este repositorio reúne prácticas y proyectos creados con distintas tecnologías del ecosistema frontend y backend, organizados modularmente para facilitar el desarrollo, la experimentación y la mejora continua.

---

## 📁 Estructura del repositorio

| Carpeta        | Descripción                                                                 |
|----------------|------------------------------------------------------------------------------|
| [`frontend`](frontend)     | Proyectos con JavaScript puro y React. UI dinámica, lógica e interacciones. |
| [`backend`](backend)       | Prácticas en PHP. Formularios, login y lógica del lado del servidor.       |
| [`sql`](sql)               | Scripts de bases de datos. Consultas, estructura y pruebas de integración. |
| [`integración`](integración) | Proyectos mixtos que combinan varias tecnologías (frontend + backend).     |
