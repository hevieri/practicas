if → Ejecuta algo solo si la condición es verdadera.

if...else → Ejecuta una cosa si es verdadera, y otra si es falsa.

switch → Compara un valor contra varios posibles casos, usando ==.

match → Devuelve un resultado según coincidencia exacta (===), más limpio y moderno.

while → Repite mientras la condición sea verdadera (puede no entrar nunca).

do...while → Igual que while, pero siempre ejecuta al menos una vez.

for → Repite una cantidad de veces predefinida, con contador incluido.

foreach → Recorre todos los elementos de un array, ideal para mostrar listas completas.
